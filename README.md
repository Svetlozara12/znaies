# ЗнайЄС — освітня платформа про Європейський Союз

Інтерактивна україномовна платформа, що пояснює, як працює ЄС, право Союзу та шлях України до членства — **простими словами і з джерелами**.

Уся платформа — це **один файл** (`index.html`), без збірки, фреймворків і залежностей. Працює офлайн (окрім архівних фото, що підвантажуються з Wikimedia Commons).

## Можливості

- **Два рівні складності** — «Просто» (для громадян) і «Глибше» (для фахівців)
- **Перемикач аудиторії** — громадянин / юрист / Україна, з блоком практичної користі
- **12 інтерактивних візуалізацій** — інституції ЄС, бюджет, кластери переговорів, чотири свободи, законодавчий процес, розширення, цінності та ін.
- **Візуальні схеми в кожній статті** + посилання на офіційні сайти інституцій ЄС
- **Історія договорів** з автентичними фото (Wikimedia Commons, вільні ліцензії)
- **50 ключових справ Суду ЄС**, провідник договорами, словник, міфи/факти
- **Тест із 8 запитань** з поясненнями
- **Спільнота й матеріали** зі збереженням у `localStorage`

## Запуск

Просто відкрийте `index.html` у браузері — або відкрийте розгорнуту версію через GitHub Pages.

## Джерела

Архівні фото — [Wikimedia Commons](https://commons.wikimedia.org/) (вільні ліцензії). Правові посилання — на офіційні ресурси ЄС: [EUR-Lex](https://eur-lex.europa.eu/), [CURIA](https://curia.europa.eu/), інституції ЄС.
